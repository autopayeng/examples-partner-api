# AUTOPAY API - Partner Usage Examples

#### This repository is provided as a usage example of the AUTOPAY API for our partners. 

For more information on our partner API, click [here](https://docs.autopay.com/#/)

> [NOTE] The payloads.zip file is password protected.  Please contact with us if you have interest in becoming a partner.  

---

#### Getting Started

`$ npm install` - install dependencies 

`$ unzip payloads.zip` - you will be prompted here for the password...enter it. 

`$ TOKEN=eyJhbGciO.... node autopayApiExample.js` - execute the service
